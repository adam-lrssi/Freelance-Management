# Freelance Management App 

![Status](https://img.shields.io/badge/Status-Work%20in%20Progress-yellow)

> **Warning:** This project is currently under active development. Features are being added and refined.


### Tech Stack 
* **Front-End:** `HTML / CSS / Tailwind`
* **Back-End:** `Python -> Flask`


### Features & Roadmap 

* `[Authentification : Client / Freealance / Admin]`
* `[Manage your missions : Add a mission / edit status / link to a client ]`
* `[Administration : Manages all users]`

### In-Progress
* **Chat page:** discussions with the client of the assignment
* **Premium account:** Make a Premium part to add more functionality

### How to run 
1. **Clone the repository**
   ```bash
   git clone httsp://github.com/adam-lrssi/python-app.git
   cd freelance-management
   ```
2. **Set up Environnement Variables:**
   > Create an venv ```bash python -m venv venv```
   > <br>
   > Install requierements ```bash pip install requirements.txt```
   > <br>
   > Run the project ```bash python main.py```
